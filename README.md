# DiaboA4

Jogo feito para a [Felpojam](https://felpojam.com/)! :3

Equipe: **Cafundó Games**

Tema: **Carimbo**

## Descrição

Um jogo em estilo _Beat-em-Up_. Você como um recem-chegado ao inferno, está tentando batalhar contra... Documentos?
Utilize seu martelo-caribo para destruir todos eles e escapar das garras do inferno!

## Controles

O jogo pode ser jogado com teclado ou controle.

No controle use o direcional esquerdo para se mover. O `A`(ou `X`) para esquivar, o `X`(ou Quadrado)
para bater e pressione os _bumpers_ para trocar entre as diferentes runas e abilidades.

No teclado use `WASD` para movimento. O `Shift` para esquivar, `Espaço` para bater e as teclas `Q` e
`E` para trocar entre as diferentes runas e abilidades
